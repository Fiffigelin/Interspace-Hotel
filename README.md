# Interspace-Hotel
Group project in school SUVNET22

Hotell Management Software
Ett system för hotel där de kan hantera bokningar och rum

● Som kund:
    [] Se vilka rum som finns lediga
    [] Boka rum
    [] Lämna en review

● Som personal:
    [] Lägga in/Ta bort rum
    [] Checka in gäst
    [] Se vilka rum som är lediga och inte
    [] Checka ut gäst
